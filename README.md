Projeto web que consome API publica do Rick e Morty para listagem de personagens, Feito com XAMPP, PHP,SQLite Javascript e Bootstrap 5.

A pagina "home" mostra personagens e suas informações. 
A pagina "personagens salvos", mostra os personagens que uma pessoa com o login feito salvou ao clicar nas imagens e selecionar as que deseja salvar.
A pagina sobre, mostra meus perfis do github e do linkedin ao clicar nos respectivos links.
A pagina login/cadastro serve para se cadastrar ou fazer seu login na aplicação.
